TP1 TEST DE VALIDATION DES ACQUIS
=================

Outils de developpement
-----------------
 * Maven 3
 * Java 8

Compilation et execution du projet
-----------------

* Installer java (jdk 8)

* Installer Maven 3 ( >3.2)

* Compilation
	cd tp1
	mvn clean pakage

* Executer
	java -jar target/tp1-1.0.jar
